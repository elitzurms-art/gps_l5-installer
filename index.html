התקנת קושחת NAVIGATE

צריבת קושחת ה-NAVIGATE ישירות מהדפדפן דרך ה-USB.

גרסה אחרונה: לא ידוע — ה-CI עוד לא רץ · קוד מקור: GitHub

הדפדפן הזה לא תומך ב-Web Serial. השתמש ב-Chrome, Edge, או Opera (גרסה 89+) על Windows / macOS / Linux / ChromeOS. Firefox ו-Safari לא נתמכים, וגם לא Chrome על iOS.

Heltec Wireless Tracker V1

ESP32-S3SX1262TFT 160×80UC6580 onboard

full node — מארח ל-GPS מוקשח חיצוני, תצוגה צבעונית, רילי תקשורת מש, GPS עזר לאיתור יחידה אבודה.

Heltec 32 V4

ESP32-S3SX1262OLED 128×64

comms-only relay — תקשורת מש + BLE + מסך OLED מונוכרום. אותם פיני תקשורת מש כמו Tracker.

Heltec 32 V3

ESP32-S3SX1262OLED 128×64

זהה ל-V4 בכל המובנים שמעניינים — תקשורת מש + BLE + מסך OLED מונוכרום.

Heltec 32 V2

ESP32 classicSX1276OLED 128×64CH340 USB

הדור הישן. ייתכן שלוחות clone דורשים BOOT+RST ידני אם ה-auto-bootloader לא עובד.

Elecrow Gateway

ESP32 classicRa-01H / SX1276ST7735S 128×160

תצוגה צבעונית, גשר GPS מוקשח + תקשורת מש. משתמש ב-polling ל-TX/RX כי DIO0 מי-מנותב על הלוח.

איך משתמשים?
  1. חבר את הלוח ל-USB.
  2. בחר את הכרטיס שמתאים לדגם שלך ולחץ Install.
  3. הדפדפן יבקש הרשאה ל-Serial port — בחר את הפורט (בדרך כלל אחד יחיד שמופיע אחרי חיבור הלוח).
  4. הצריבה תרוץ אוטומטית. הקושחה מוגדרת ל-868 MHz / SF9 / BW250 / sync word 0x34, ומפרסמת BLE בשם NAVIGATE-XXXX (4 הקסה תחתונים של ה-MAC).
  5. אם זה V2 או Elecrow ישן והצריבה לא מתחילה: החזק BOOT בזמן שאתה לוחץ ושחרר את RST, ואז לחץ Install שוב.

חיווט GPS מוקשח חיצוני שונה פר-לוח — ראה את ה-README של הקושחה לפינים המדויקים.

פתרון בעיות מהירות
  • Web Serial לא נתמך — חייב Chrome / Edge / Opera על דסקטופ. Android מהדפדפן Chrome כן עובד; iOS אף פעם לא.
  • הצריבה תקועה ב-Connecting… — סגור את כל ה-serial monitors / Arduino IDE / PlatformIO וכל מה שמחזיק את הפורט.
  • Wrong chip detected — בחרת לוח לא נכון. ESP32 vs ESP32-S3 שונים — ה-Heltec V2/Elecrow הם ESP32 קלאסי, ה-V3/V4/Tracker הם ESP32-S3.
  • Bootloop אחרי flash — בדרך כלל אומר שגודל ה-flash במאניפסט לא תואם ללוח (4MB vs 8MB). פתח issue ב-GitHub.
